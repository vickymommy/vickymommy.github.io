# 如何執行「文章下載腳本」

不需要安裝任何東西，用 **Windows 內建的 PowerShell** 就能跑。
（檔案 `下載文章.mjs` 是 Node 版備用；沒裝 Node 的話，請用下面的 PowerShell 版 `下載文章.ps1`。）

## 最簡單的方式：複製這一行去執行

1. 在開始選單搜尋 **PowerShell**，打開它。
2. 把下面這一整行貼上，按 Enter：

```
powershell -ExecutionPolicy Bypass -File "C:\Users\justi\Desktop\個人學習\Vicky Mommy 玲玲_網站規畫\網站專案\下載文章.ps1"
```

就會開始一篇一篇下載，畫面會顯示每篇 OK／X。82 篇＋圖片大約幾分鐘。

> 也可以：在檔案總管打開「網站專案」資料夾 → 對著 `下載文章.ps1` 按右鍵 →「用 PowerShell 執行」。

## 它會做什麼

- 讀 `data\articles.json` 的文章清單。
- 逐篇下載 vocus 原文，擷取正文（自動去掉選單、推薦區等雜訊）。
- 內文圖片與封面圖**下載到站內** `assets\images\`，並把圖片改成本機路徑。
- 寫好每篇 `articles\<id>.html`。

## 跑完之後

1. 回到這個對話跟我說「**跑完了**」（或貼上最後的統計：新增 N 篇／失敗 N 篇）。
2. 我會在這邊**重新產生資料檔**（`data\articles.js`）並檢查所有內文與圖片是否正確呈現、微調排版。
3. 接著進入後台（PagesCMS）建置與部署。

## 若有幾篇失敗（顯示 X）

多半是該篇格式特殊或為會員文。可以再執行一次（已完成的會自動略過）。
仍失敗的，把那幾篇的網址貼給我，我個別處理。

## 常見問題

- 出現紅字「執行原則」被擋：請用上面那一行含 `-ExecutionPolicy Bypass` 的指令，就不會被擋。
- 中途中斷沒關係，重跑會接續（已完成的會略過）。
